# Ícones PWA

Para completar a configuração do PWA, você precisa criar os seguintes ícones na pasta `public/`:

- `pwa-192x192.png` - Ícone 192x192 pixels
- `pwa-512x512.png` - Ícone 512x512 pixels

Você pode usar ferramentas online como:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

Ou criar manualmente usando qualquer editor de imagens.

**Nota:** O PWA funcionará mesmo sem esses ícones, mas é recomendado adicioná-los para uma melhor experiência do usuário.
